Week 7: Diffie Hellman Key Exchange

Content

Diffie-Hellman Key Exchange (DH) is a fundamental protocol in modern cryptography used for establishing a shared secret between two parties over an insecure channel.

In this lecture, we will delve into the computation of primitive roots, understand the discrete logarithm problem (DLP), explore the Diffie-Hellman protocol with a numerical example, and summarize the key concepts learned.

Learning Outcome

By the end of this lecture, students should be able to:

  • Identify the application of Diffie-Hellman Key Exchange (DH) for secure communication.
  • Perform simple computations to generate shared symmetric keys using DH.

Primitive Root

A primitive root of a prime number 𝑝 is an integer 𝑔 such that every integer from 1 to 𝑝-1 can be expressed as a power of 𝑔 modulo 𝑝.

To find a primitive root 𝑔 for a prime 𝑝:

  1. Choose a prime number 𝑝.
  2. Select a generator 𝑔 such that 𝑔 < 𝑝.
  3. Compute 𝑔^𝑖 mod 𝑝 for values of 𝑖 from 1 to 𝑝-1.
  4. If all results are distinct, then 𝑔 is a primitive root of 𝑝.

Discrete Logarithm Problem (DLP)

The discrete logarithm problem involves finding an integer 𝑖 such that 𝑔^𝑖 ≡ 𝑎 (mod 𝑝) for given 𝑎, 𝑔, and 𝑝. This problem is computationally hard to solve, especially for large prime numbers 𝑝.

Example:

  • 𝑔^4 ≡ 5 (mod 7)
  • 𝑔^5 ≡ 4 (mod 7)

The Diffie-Hellman Protocol

The Diffie-Hellman Key Exchange Protocol is not an encryption/decryption algorithm but rather a method for securely exchanging cryptographic keys over an insecure channel.

Steps involved:

  1. Publicly share information about a large prime number 𝑝 and a generator 𝑔.
  2. Both parties select secret keys: 𝑎 for Alice and 𝑏 for Bob.
  3. Combine secret keys with public information to generate shared values 𝑌𝐴 and 𝑌𝐵.
  4. Exchange the combined values.
  5. Compute the shared secret key.

Summary

Diffie-Hellman Key Exchange enables secure communication over insecure channels by allowing parties to generate a shared secret key without prior communication. By leveraging the computational hardness of certain mathematical problems, such as the discrete logarithm problem, DH provides a robust mechanism for key exchange.
